Here is how you can make your own rows
< div class = "row" >
< div class="col-lg-6" > lorem < /div >
< div class="col-lg-6" > ipsum < /div >

Notice how each row has 12 columns. The number after col-lg- specifies the size of the column. Make sure they total up to 12.

Here is how you can make your own tables
< table border = "SIZE" class="table TABLE_TYPE" >
  < thead >
   < tr >
    < th >Lorem< /th >
    < th >Ipsum< /th >
   < /tr >
  < /thead >
  < tbody >
   < tr >
    < td >orci< /td >
    < td >nulla< /td >
   < /tr >
  < /tbody >
< /table >
 SIZE will be the size of the border around the table; insert an integer.
TABLE_TYPE is the type of table you wish to have. We are using table-hover which shades the row your mouse is over gray.
You can also choose from table-striped, table-bordered, and table-condensed.
